Tesla Stock Price Analysis

This project performs Exploratory Data Analysis (EDA) on Tesla stock market data to understand historical price movements and trends. The analysis uses Python libraries to retrieve stock data, visualize price patterns, and analyze moving averages to observe short- and long-term market behavior.

Objective

The goal of this project is to explore Tesla's stock performance and identify patterns using statistical analysis and visualization techniques.

Key Analysis Performed
-  Downloading historical stock data using yfinance
-  Data preprocessing and cleaning
-  Visualization of stock price trends
-  Calculation of moving averages (10, 20, 50 days)
-  Time series analysis of price fluctuations
-  Comparison of short-term and long-term trends

Results

The analysis helps visualize Tesla's historical stock behavior and demonstrates how moving averages can highlight market trends and potential momentum shifts.

How to Run the Project
1. Clone the repository
   git clone https://github.com/AmanGoyal49/SP-500-EDA.git
2. Install required libraries
   pip install pandas numpy matplotlib seaborn yfinance
3. Run the Jupyter Notebook
   jupyter notebook
